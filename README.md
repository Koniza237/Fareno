## Description

FARENO est une solution d'automatisation avancée pour WhatsApp, développée par FARENO-TECH. Ce bot intègre des fonctionnalités d'intelligence artificielle (IA) pour la modération de groupes, la récupération de messages, la planification d'envois, l'analyse de conversations, le téléchargement de médias, et bien plus. Idéal pour les utilisateurs et les administrateurs de groupes WhatsApp qui souhaitent optimiser leurs interactions quotidiennes.

Le projet est basé sur une page web statique (index.html) qui sert de vitrine promotionnelle, décrivant les commandes du bot, les fonctionnalités, et les plans d'abonnement. Le bot lui-même est une application automatisée pour WhatsApp, avec des intégrations IA comme Bing, DALL-E, GPT et Grok.

## Fonctionnalités

Voici un aperçu des principales fonctionnalités du bot FARENO, accessibles via des commandes simples comme `.vv`, `.delete`, etc. :

- **Anti Vue Unique et Récupération de Messages** :
  - `.vv` : Récupère les messages en vue unique sans marquer comme lus.
  - `.delete` : Récupère les messages supprimés.

- **Gestion de Groupes** :
  - `.add <numéro>` : Ajoute un membre.
  - `.kick <@membre>` : Expulse un membre.
  - `.promote <@membre>` : Promouvoir en admin.
  - `.mute <@membre>` : Mute un membre.
  - `.tag` : Tag tous les membres.

- **Planification et Analyse** :
  - `.setschedule <message> <date>` : Planifie l'envoi de messages.
  - `.msgstats` : Analyse les stats de conversations.

- **Gestion des Statuts** :
  - `.setstatus <texte>` : Définit un statut.
  - `.status` : Affiche les statuts.

- **Téléchargement de Médias** :
  - `.story <@utilisateur>` : Télécharge les stories.
  - `.video <lien YouTube>` : Télécharge des vidéos.
  - `.insta <lien Instagram>` : Télécharge du contenu Instagram.

- **Filtres Anti-Spam** :
  - `.antispam` : Active l'anti-spam.
  - `.antilink` : Bloque les liens.
  - `.antiword <mot>` : Bloque des mots spécifiques.

- **Création de Contenu** :
  - `.sticker` : Crée des stickers.
  - `.circle` : Convertit en cercle.
  - `.mp4` : Convertit en MP4.
  - `.textmaker <effet> <texte>` (ex. `.neon`, `.glitch`, `.marvel`) : Génère des textes stylisés.

- **Jeux et Interactions** :
  - `.tictactoe` : Jeu de morpion.
  - `.vote` : Système de vote.

- **Intégration IA** :
  - Bing, DALL-E, GPT, Grok pour des réponses intelligentes et des générations d'images.

## Installation

Ce dépôt contient principalement une page web statique (index.html). Pour l'utiliser localement :

1. Clonez le dépôt :
   ```
   git clone https://github.com/Koniza237/Fareno.git
   ```
2. Ouvrez `index.html` dans un navigateur web.

Pour déployer sur un serveur (comme Render), téléchargez les fichiers et configurez un site statique.

**Note** : Le bot WhatsApp lui-même n'est pas inclus dans ce dépôt ; il s'agit d'un service payant. Contactez-nous pour l'installation et la configuration.
